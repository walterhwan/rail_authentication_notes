# Rails Commands

### Creating a new rails project in postgre
```bash
rails new app_name --database=postgresql
```
### Generate Migration
```bash
rails g migration CreateObjects
```
### Generate Controllers
```bash
rails g controller Objects
```
this will generate objects_controller.rb
